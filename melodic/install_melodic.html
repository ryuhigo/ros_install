Raspberry Pi にROS(Melodic)をインストールする方法


    ROSのインストール手順

    • Ubuntuのインストール

    • ROSのインストール


    1. Ubuntuのインストール

    • Ubuntu(18.04)のイメージファイルをダウンロード

    • イメージファイルの書き込み (以下はbalenaEtcherを用いた方法)

      • ・balenaEtcherのダウンロードページに移動してダウンロードする．

      ・パソコンにSDカードを差し，balenaEtcherを起動する．

      ・"Falsh from file"にダウンロードした"Ubuntu18.04"のイメージファイルを選択し， "Select target"にSDカードを選択した後に"Flash!"を押す．


    2. ROSのインストール

    • ラズパイのログイン設定(ラズパイにSDカードを挿し，電源を入れ以下のように入力する)

      • ubuntu login:ubuntu
      Password:ubuntu
      (Current) UNIX password:ubuntu
      Enter new UNIX password:新しいパスワードを入力
      Retype new UNIX password:新しいパスワードを再入力

    • ROSのダウンロード先のアドレスをsources.lstに追加

      • ・$ sudo sh -c 'echo "deb http://packages.ros.org/ros/ubuntu $(lsb_release -sc) main" > /etc/apt/sources.list.d/ros-latest.list'

    • 鍵を設定し,リポジトリに追加

      • ・$ sudo apt-key adv --keyserver 'hkp://keyserver.ubuntu.com:80' --recv-key C1CF6E31E6BADE8868B172B4F42ED6FBAB17C654

    • パッケージリストの更新

      • ・$ sudo apt update

    • ROSのインストール

      • ・$ sudo apt install ros-melodic-ros-base

    • ROSの環境変数の設定

      • ・$ echo "source /opt/ros/melodic/setup.bash" >> ~/.bashrc

      ・$ source ~/.bashrc

    • 依存関係をインストール

      • ・$ sudo apt install python-rosdep python-rosinstall python-rosinstall-generator python-wstool build-essential

    • rosdepを初期化

      • ・$ sudo apt install python-rosdep

      ・$ sudo rosdep init

      ・$ rosdep update

    • ワークスペースの作成

      • ・$ mkdir -p 〜/catkin_ws/src

      ・$ cd 〜/catkin_ws

      ・$ catkin_make

    • ワークスペースをインストール環境に配置

      • ・$ echo "source ~/catkin_ws/devel/setup.bash" >> ~/.bashrc

      ・$ source ~/.bashrc
